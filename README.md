The penticon Gem
============
> An implementation of Penticons in [ruby](https://www.ruby-lang.org/en/) programming language.

Identicons similar to GitHub's contribution activity calendar.


```go
require 'penticon'

arg = "Penticons"
p = Penticon.Uri_image(arg)
print p
```

####Installation
```go
gem install penticon
```
